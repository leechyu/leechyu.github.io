🔒 隐私政策

最后更新日期：2026-06-14

1. 概述

Knot（以下简称"本应用"）重视您的隐私。本隐私政策说明了我们如何收集、使用、披露和保护您的个人信息。请仔细阅读本政策，以了解我们在数据处理方面的做法以及您的权利。

本应用面向全球全年龄段用户提供服务，我们遵守适用的隐私法律，包括但不限于美国《加利福尼亚州消费者隐私法案》(CCPA/CPRA)、《儿童在线隐私保护法》(COPPA)，以及欧盟《通用数据保护条例》(GDPR) 等其他适用法规。

2. 我们收集的信息

2.1 您主动提供的信息

数据类别具体内容用途
账户信息邮箱地址、加密后的登录密码用户身份识别与登录认证
第三方登录信息Apple ID 或 Google 账户的 identity token（仅用于验证身份，不存储第三方密码）Apple Sign-In / Google Sign-In 快捷登录
个人资料显示名称（可自行设置）账户个性化展示
笔记内容您创建的笔记正文、标签、附件（图片、音频）提供核心笔记服务
语音数据录音文件及语音转文字结果（转换在设备端完成）语音笔记功能
提醒设置您为笔记设置的提醒时间到时间发送本地通知

2.2 自动收集的信息

数据类别具体内容用途
设备信息设备名称、平台类型（Android/iOS）多设备管理与安全审计
关联向量笔记内容经 embedding 模型计算的 384 维向量自动发现笔记间的关联关系（完全自动化，不涉及人工审阅）
崩溃日志应用崩溃时的错误堆栈（通过 Sentry SDK 收集，不包含笔记内容）提升应用稳定性
版本历史笔记编辑过程中自动保存的历史版本版本回退功能

2.3 我们不收集的信息

  • 精确地理位置：我们不收集 GPS 定位数据。
  • 通讯录：我们不访问您的设备通讯录。
  • 浏览历史：我们不跟踪您在其他应用或网站上的活动。
  • 生物识别信息：我们不收集指纹、面容等生物特征。

3. 数据存储位置与安全

  • 所有数据存储在我们自托管的服务器上（PostgreSQL + pgvector 数据库 + 云对象存储）。
  • 中国版用户的服务器部署于中国大陆，其余用户部署于美国，就近存储以降低延迟。
  • 传输安全：所有 API 通信使用 HTTPS (TLS 1.2+) 加密。
  • 存储安全：密码使用 bcrypt 加盐哈希存储；附件使用私有存储桶 + 签名 URL 访问。
  • 访问控制：所有 API 请求需携带 JWT token 验证身份，支持 token 自动刷新。
  • 版本记录：笔记自动保存历史版本，可在版本抽屉中查看和恢复。

4. 数据如何使用

  • 仅为您本人服务：您的笔记数据和关联向量仅用于为您提供笔记管理、自动关联发现、全文模糊搜索和知识图谱功能。
  • 自动关联：系统使用 embedding 模型将笔记内容转换为向量以自动发现相关笔记——此过程完全自动化，不涉及人工审阅。
  • 本地提醒：提醒通知由设备本地调度，提醒数据不会发送给第三方推送服务。
  • 不用于广告：我们不使用您的数据投放广告或构建用户画像。
  • 不用于自动化决策：我们不会基于您的数据做出对您产生法律或类似重大影响的自动化决策。

5. 数据共享与披露

5.1 与第三方共享

我们不出售您的个人信息（"出售"定义参见 CCPA）。除非以下情况，我们不会将您的个人数据分享给任何第三方：

共享情形说明
法律要求依法向执法机构或监管机关披露必要信息
服务提供商向受合同约束的服务提供商（如云存储）披露必要数据以运营服务
经您同意在获得您明确同意的情况下向特定第三方披露

5.2 第三方 SDK 与服务

本应用使用以下第三方 SDK，它们有各自的隐私政策：

7.1 加利福尼亚州居民特定权利

如果您是加利福尼亚州居民，根据 CCPA/CPRA，您还有权：

  • 请求披露我们收集的个人信息类别和具体内容（申请后 45 天内回复）
  • 请求披露个人信息的来源类别、收集或出售的商业目的，以及与之共享的第三方类别
  • 限制对敏感个人信息的使用和披露（我们目前不收集 CCPA 定义的敏感个人信息）

如需行使上述权利，请通过第 12 节中的联系方式联系我们。我们将在收到可验证的请求后 45 天内回复。

7.2 授权代理人

您可以指定授权代理人代表您提交 CCPA 请求。代理人需提供您的书面授权证明。

8. 儿童隐私 (COPPA)

本应用面向全年龄段用户开放。13 岁以下儿童可在父母或法定监护人的指导和同意下使用本应用。我们遵守《儿童在线隐私保护法》(COPPA) 的规定。如果您认为我们可能持有 13 岁以下儿童的信息且其监护人未提供适当同意，请联系我们。

9. 请勿追踪 (Do Not Track)

我们的服务不会响应浏览器的"请勿追踪"(DNT) 信号，因为我们不跨网站或应用跟踪用户行为。

10. 数据泄露通知

如果发生涉及您个人信息的数据泄露事件，我们将在法律要求的时限内通知您和相关监管机构。

11. 政策更新

本隐私政策可能不时更新。重大变更时，我们将通过以下方式告知您：

  • 应用内弹窗通知
  • 如变更涉及新的数据收集用途，我们将重新获取您的同意

继续使用本应用即表示接受更新后的隐私政策。
